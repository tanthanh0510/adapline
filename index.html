Adaptive linear neurons¶

Adaptive linear neurons (Adaline ) l mt loi single-layer neural network (NN): c xut bn bi Bernard Widrow v Tedd Hoff ch mt vi nm sau thut ton perceptron. Adapline nh ngha cc khi nim chnh v ti u hm chi ph, t nn tng cho cc thut ton machine learning tin tin hn phn loi sau ny nh logistic regression, support vector machines v regression models. S khc bit chnh gia Adaline v perceptron l cc trng s c cp nht da trn activation function thay v unit step function nh trong perceptron. Trong Adaline, activation function ny, $\phi(z)$ , ch n gin l nh ngha hm net_input, do : $$\phi(w^Tx)=w^Tx $$ Mc d activation function c s dng hc cc trng s, nhng Adaline vn s dng hm ngng (threshold function) a ra d on cui cng, tng t nh unit step function perceptron.

Tối ưu hàm chi phí (cost functions) bằng Gradient Descent (GD)¶

1. Hàm chi phí (cost functions)¶

Mt trong nhng thnh phn chnh ca vic hc c gim st (supervised learning) l nh ngha hm mc tiu ti u ha trong qu trnh hc. trong adaline, ta nh ngha hm mc tiu l hm chi ph $J$ hc cc trng s l tng bnh phng li <sum of squared errors (SSE)> gia kt qu tnh ton v nhn thc t.$$J(w)=\frac{1}{2}\sum\Big((y^{(i)}-\phi(z^{(i)})\Big)^2$$ $\frac{1}{2}$ c thm vo d dng ly c dc ca hm chi ph hoc tn tht i vi cc trng s. u im ca activation function ny l n lin tc, tri ngc so vi unit step function iu ny lm cho hm chi ph n khc bit. V c bit hm chi ph ny li, v th ta s s dng mt phng php ph bin ti u chnh l Gradient Descent (GD)

2. Gradient Descent (GD)¶

GD l phng php tm gi tr nh nht (i khi l ln nht ) ca hm s. Hng tip cn ca thut ton ny l xut pht t mt im m chng ta coi l gn vi nghim ca bi ton, sau dng mt php ton lp tin dn n im cn tm, tc n khi o hm gn vi 0.

Gi s ta cn tm gi tr nh nht cho hm $ f(\theta)$. u tin ta t $\theta$ ti mt im bt k. Sau cp nht $ \theta$ theo cng thc: $$\theta_{i+1}=\theta_i-\eta\nabla_\theta f(\theta_i)$$

  • V d hm s $f(x,y)=(x^2+y-7)^2+(x-y+1)^2$ s c hai im cc tiu ti $(2,3)$ v $(-3,-2)$, di y l hnh minh ha khi s dng gradient descent tm cc tiu, ty vo im khi to m ta s tm c cc kt qu khc nhau sau mi epochs

Quy tc cn nh: lun lun i ngc hng vi o hm.

* Vì sao phải đi ngược hướng với đạo hàm ?¶

V d: ta cn ti u (tm cc tiu ) hm chi ph $J(w)$ vi $w$ l mt vect. o hm ca $J(w)$ theo vect $w$ s c nh ngha l: $\nabla_w J(w) = \begin{bmatrix} \frac{\partial f(w)}{\partial w_1} \\ \vdots \\ \frac{\partial f(w)}{\partial w_n} \end{bmatrix}$

  • Ta cp nht trng s ca $w$ ln thm mt khong $\eta x$ th hm chi ph mi ta t c l $J(w+\eta x)$ vi $\eta$ l tc hc (tc bc ).
  • iu ta cn chng minh l: $J(w) > J(w+\eta x)$ hay $J(w+\eta x) - J(w)< 0$
  • Ta p dng khai trin Taylor : $f(x)=f(a)+\frac{f'(a)}{1!}(x-a)+ \frac{f^{(2)}(a)}{2!}(x-a)^2 + \dots$
  • Ta c:$$J(w+\eta x)=J(w)+\frac{\nabla J(w)}{1!}\big((w+\eta x)-w\big)+ \frac{\nabla^2 J(w)}{2!}\big((w+\eta x)-w\big)^2 + ... $$ $$ \iff J(w+\eta x)=J(w)+\eta x\nabla J(w)+ \frac{\eta^2}{2!}x\nabla^2 J(w) + ... \quad\quad\quad\quad\quad(1)$$
  • Vi $\eta$ nh th $\eta^2$ s tin v $0$ th biu thc $(1)$ s xp x bng: $$ J(w+\eta x)\approx J(w)+\eta x\nabla J(w) \iff J(w+\eta x)-J(w)=\eta x\nabla J(w)$$
  • Gi $\alpha$ l gc to vi $x$ v $\nabla J(w)$ ta c: $$\cos(\alpha)=\frac{x\nabla J(w)}{|x||\nabla J(w)|}$$
  • Mc khc ta c $$-1 \leq \cos(\alpha) \leq 1\iff -1 \leq \frac{x\nabla J(w)}{|x||\nabla J(w)|} \leq 1 \quad\quad\quad (2)$$
  • t $k=|x||\nabla J(w)| > 0$, th t $(2)$ ta s c: $$-\eta k \leq \eta x\nabla J(w) \leq \eta k \iff -\eta k \leq J(w+\eta x)-J(w) \leq \eta k$$
  • iu ta mun l hm chi ph s m hoc nh nht, t biu thc trn ta thy $min\big(J(w+\eta x)-J(w)\big)=-\eta k$ khi $\cos(\alpha)=-1$,iu ny c ngha l gc to bi $x$ v $\nabla J(w)$ l $180^o$ tc l hng ca $x$ phi ngc vi hng ca $\nabla J(w)$ ### 3. Ti u hm chi ph bng GD Ta c: $$J(w)=\frac{1}{2}\sum\Big((y^{(i)}-\phi(z^{(i)})\Big)^2$$ p dng GD ta c biu thc cp nht ca $w$ l : $w:=w+\nabla w$. Trong $\nabla w$ c nh ngha l dc m nhn vi t l hc tp, $\eta$: $$\nabla w = -\eta \nabla J(w)$$ tnh ton, ta cn tnh o hm ring ca hm chi ph theo tng trng s, $w_j$: $$\frac{\partial J}{\partial w_j} = -\sum_i \big(y^{(i)} - \phi (z^{(i)}) \big)x_j^{(i)}$$ V vy c th vit cng thc cp nht ca $w_j$ l: $$\nabla w_j = -\eta\frac{\partial J}{\partial w_j}= \eta\sum_i \big(y^{(i)} - \phi (z^{(i)}) \big)x_j^{(i)}$$ V cp nht tt c cc trng s nn cng thc cp nht s l: $$ w:= w+\nabla w$$

import numpy as np
class AdalineGD(object):
    def __init__(self, eta=0.01, n_iter=50, random_state=1):
        self.eta=eta
        self.n_iter=n_iter
        self.random_state=random_state
    def fit(self,X,y):
        rgen = np.random.RandomState(self.random_state)
        self.w_ = rgen.normal(loc=0.0, scale=0.01,size=1 + X.shape[1])
        self.cost_ = []
        for i in range(self.n_iter):
            net_input = self.net_input(X)
            output = self.activation(net_input)
            errors = (y - output)
            self.w_[1:] += self.eta * X.T.dot(errors)
            self.w_[0] += self.eta * errors.sum()
            cost = (errors**2).sum() / 2.0
            self.cost_.append(cost)
        return self
    def net_input(self,X):
        return np.dot(X, self.w_[1:]) + self.w_[0]
    def activation(self, X):
        return X
    def predict(self, X):
        return np.where(self.activation(self.net_input(X)) >= 0.0, 1, -1)

Improving gradient descent through feature scaling¶

Gradient descent l mt trong nhiu thut ton c hng li t feature scaling. phn ny ta s s dng mt phng thc ca feature scaling c gi l tiu chun ha (standardization) N s a d liu ca ta v dng phn phi chun vi vi phng sai n v (c lch chun bng 1) v c trung bnh gn 0.Tuy nhin n khng lm cho tp gc thnh phn phi chun, tp gc vn khng i v phn phi d liu, N dch cc im d liu v dng m trung bnh ca n v tr 0 v c phng sai l 1.

V d, chun ha tnh nng th j, chng ta ch cn tr trung bnh mu, $\mu_j$ t mi v d o to v chia n cho lch chun ca n, $\sigma_j$: $$x_j^{'} = \frac{x_j-\mu_j}{\sigma_j}$$ y, $x_j$ l vect bao gm cc gi tr tnh nng th j ca tt c v d ca d liu o to, n , v k thut chun ha ny c p dng cho tng tnh nng, j, trong tp d liu. Mt trong nhng l do ti sao tiu chun ha gip hc tp gim dn l v trnh ti u ha phi tri qua t bc hn tm ra gii php tt hoc ti u (ti thiu chi ph), nh c minh ha trong hnh di y, trong cc cu hnh con biu th b mt chi ph nh mt hm ca hai trng s m hnh trong bi ton phn loi hai chiu:

    plt.ylim(xx2.min(), xx2.max())

    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(x = X[y == cl, 0], y = X[y == cl, 1], alpha = 0.9, c = colors[idx], marker = markers[idx], label = cl, edgecolor = 'black')

Large-scale machine learning and stochastic gradient descent¶

phn trc ta s dng phng php Gradient Descent hay cn gi l batch gradient descent v n c tnh ton d trn ton b tp d liu. iu ny s tn km khi ta lm vic vi nhng b d liu ln, v th ta cn nh gi li ton b tp d liu o to mi khi chng ta thc hin mt bc v mc ti thiu. Mt thay th ph bin cho thut ton batch gradient descent l stochastic gradient descent (SGD), i khi cn c gi l gim dn dc trc tuyn hoc lp li (iterative or online gradient descent). Thay v cp nht cc trng s da trn tng s li tch ly trn tt c cc v d o to, $x^{(i)}$: $$\Delta w = \eta \sum_i \big(y^{(i)}-\phi(z^{(i)})\big)x^{(i)}$$ Chng ta cp nht cc trng s tng dn cho mi v d o to: $$\eta \big(y^{(i)}-\phi(z^{(i)})\big)x^{(i)}$$ Mc d SGD c th c coi l GD, nhng n thng t tc hi t nhanh hn nhiu do cp nht trng lng thng xuyn hn. V mi o hm c tnh ton da trn mt v d duy nht. c c kt qu tha mn thng qua SGD, iu quan trng l phi trnh by d liu theo th t ngu nhin Ngoi ra, ta cn xo trn tp d liu cho mi epochs ngn chn chu k.

Mt u im khc ca SGD l c th s dng n cho vic online learning. Trong online learning, m hnh ca ta s c o to nhanh chng khi c d liu o to mi.

class AdalineSGD(object):
    def __init__(self, eta=0.01, n_iter=10,
        shuffle=True, random_state=None):
        self.eta = eta
        self.n_iter = n_iter
        self.w_initialized = False
        self.shuffle = shuffle
        self.random_state = random_state
    def fit(self, X, y):
        self._initialize_weights(X.shape[1])
        self.cost_ = []
        for i in range(self.n_iter):
            if self.shuffle:
                X, y = self._shuffle(X, y)
            cost = []
            for xi, target in zip(X, y):
                cost.append(self._update_weights(xi, target))
            avg_cost = sum(cost) / len(y)
            self.cost_.append(avg_cost)
        return self
    def partial_fit(self, X, y):
        if not self.w_initialized:
            self._initialize_weights(X.shape[1])
        if y.ravel().shape[0] > 1:
            for xi, target in zip(X, y):
                self._update_weights(xi, target)
        else:
            self._update_weights(X, y)
        return self
    def _shuffle(self, X, y):
        r = self.rgen.permutation(len(y))
        return X[r], y[r]
    def _initialize_weights(self, m):
        self.rgen = np.random.RandomState(self.random_state)
        self.w_ = self.rgen.normal(loc=0.0, scale=0.01,size=1 + m)
        self.w_initialized = True
    def _update_weights(self, xi, target):
        output = self.activation(self.net_input(xi))
        error = (target - output)
        self.w_[1:] += self.eta * xi.dot(error)
        self.w_[0] += self.eta * error
        cost = 0.5 * error**2
        return cost
    def net_input(self, X):
        return np.dot(X, self.w_[1:]) + self.w_[0]
    def activation(self, X):
        return X
    def predict(self, X):
        return np.where(self.activation(self.net_input(X)) >= 0.0, 1, -1)
